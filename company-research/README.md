# Company Research Plugin

企業URLを渡すと、知りたい情報をインタラクティブに選択しながら調査・まとめてくれるプラグイン。

## 特徴

- **インタラクティブな情報選択** — URLを渡すと何を調べるか確認してから調査開始
- **目的に合わせた調査** — 必要な情報だけを選んで効率的にリサーチ
- **構造化レポート** — 選択した項目のみを見やすい形式で出力
- **ウェブ検索ベース** — 追加ツール不要でそのまま使える

## 使い方

### スラッシュコマンド

```
/company-research:company-intel https://example.com
```

### 自然言語

- 「https://example.com について調べて」
- 「この会社の情報をまとめて: https://example.com」
- 「https://example.com の企業リサーチをして」

### 収集できる情報

| カテゴリ | 内容 |
|----------|------|
| 会社概要・事業内容 | 何をしている会社か、業界、規模、設立年 |
| 経営陣・キーパーソン | CEO・役員・創業者の経歴 |
| 財務・資金調達 | 調達履歴、投資家、売上推定 |
| 最新ニュース・動向 | 直近のプレスリリース・メディア露出 |
| 採用・組織動向 | 求人情報、採用トレンド |
| 製品・サービス詳細 | プロダクト・サービスの詳細、価格帯 |
| 競合・市場ポジション | 主要競合、差別化ポイント |
| 技術スタック | 使用技術・フレームワーク |

```
├── commands/
│   └── company-intel.md         # スラッシュコマンド
└── skills/
    └── company-intel/
        └── SKILL.md             # メインスキル
```

## CONNECTORS.md

このプラグインはウェブ検索のみで動作します。外部コネクターは不要です。

将来的に以下のコネクターで機能強化が可能:

| コネクター | 追加できる情報 |
|-----------|---------------|
| **Crunchbase / PitchBook** | 正確な資金調達データ、投資家詳細 |
| **LinkedIn** | 経営陣の詳細プロフィール |
| **StackShare** | 技術スタックの詳細 |
