## インシデント事例

### 記入例: 2025-10-15: キャッシュ無効化漏れによるデータ不整合

**影響範囲：** backend — ユーザープロフィール表示
**根本原因：** プロフィール更新 API 改修時に Redis キャッシュの無効化処理を追加し忘れた
**変更パターン：** データ更新系 API の変更 / キャッシュ戦略の変更
**再発防止テスト：**
- データ更新後に画面をリロードして最新データが表示されること
- 複数タブで開いている場合も更新が反映されること
