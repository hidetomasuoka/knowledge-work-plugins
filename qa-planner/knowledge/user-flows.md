## フロー詳細

### ユーザー登録

```
1. トップページ → 「登録」ボタン
2. メールアドレス・パスワード入力
3. 認証メール受信 → リンククリック
4. プロフィール初期設定
5. ダッシュボードへ遷移
```

**関係するサービス/API：**
- `POST /api/v1/users` (backend)
- メール送信サービス

**テスト時の注意：** STG 環境ではメールはモックのため実際には送信されない

---

### ログイン / ログアウト

```
1. ログインページでメールとパスワードを入力
2. セッション発行 → ダッシュボードへ
3. ログアウトボタン → セッション破棄 → トップページへ
```

**関係するサービス/API：**
- `POST /api/v1/sessions` (backend)
- Redis（セッション管理）

## カスタマイズ方法

1. フロー一覧に自社の主要機能を追加し、重要度を設定する
2. 各フローの操作ステップを実際のフローに書き換える
3. 関係するサービス/API を実際のものに書き換える
4. テスト時の注意点（環境差異など）を追記する
